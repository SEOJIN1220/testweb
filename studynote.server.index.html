Study Note

[6장] 디스크 관리
* 쿼터 : 파일 시스템마다 사용자나 그룹이 생성할 수 있는 파일의 용량과 개수를 제한하는 것
[8장] 원격지 시스템 관리
* 텔넷 : 인터넷을 통하여 원격지의 호스트 컴퓨터에 접속할 때 지우너되는 인터넷 표준 프로토콜
* SSH : secure shell의 약어. PGP와 마찬가지로 공개 키 방식의 암호 방식을 사용하여 원격지 시스템에 접근하여 암호화된 메시지를 전송할 수 있는 시스템

[9장] 네임서버
* IP 주소를 얻는 흐름
* 캐시전용네임서버
* 마스터 네임서버

[10장] 메일 서버
* 나오는 명령어들 : apt -y install sendmail, apt -y install bind9 bind9utils, ufw disable, server 192.168.111.100, mail.naver.com, mail.daum.net, systemtl restart NetworkManager, netsh interfaxe ip set dns Ethernet0 static 192.168.111.100, apt -y install dovecot-pop3d, makemap hash /etc/mail/access < /etc/mail/access

[11장] DB 서버
* 원통 모양의 데이터베이스를 관리하기 위한 소프트웨어
* 역할 : 데이터베이스 이름 조회, 사용할 데이터베이스 지정, 데이터베이스 생성, 데이터베이스 삭제

[12장] 웹 서버
* 웹 브라우저 클라이언트로부터 http 요청을 받아들이고 html 문서와 같은 웹 페이지를 반환하는 컴퓨터 프로그램
* 부속된 모듈을 이용하면 네트워크 간의 프록시 연결이나 ssl 설정 쉽게 가능
* 정적 컨텐크를 요청 받으면 즉시 응답

[13장] FTP 서버
* 파일을 하나의 장치에서 다른 장치로 전송하는 소프트웨어 애플리케이션. FTP 주소를 가지고 있는 FTP 연결을 수신하는 데 사용되는 컴퓨터.

[15장] SAMBA 서버
* 운영체제가 다른, 즉 윈도우와 리눅스 사이의 접근을 쉽게하기 위해 도와주는 프로그램
* 윈도우에서 리눅스 서버에 탐색기처럼 접근하여 파일을 쉽게 읽고 쓸 수 있음
