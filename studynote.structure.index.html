"자료 구조"

기말고사 필기노트

Study Note

[5장] 스택
* 구조 : 입력과 출력을 한쪽에서만 가능하게 만든 구조
* 활용 : 먼저 입력되면 아래에 저장되고, 나중에 입력되면 위에 저장되므로 입력순서와 출력순서를 변경시킬 수 있음
* 후입 선출 : 가장 나중에 입력된 데이터가 가장 먼저 출력 됨(먼저 들어온 데이터가 아래에 저장 -> 가장 늦게 사용 가능)
* 스택의 주요 연산 : 삽입, 삭제 (배열 스택에서의 입출력 동작)
* 기타 연산 : is_empty(s), is_full(s), create(), peeks(s)

[6장] 큐
* 큐 : 먼저 들어온 데이터가 먼저 나가는 자료구조
* 동작 패턴 : 선입선출 / 구조 : 배열, 연결리스트
* 삽입과 삭제는 FIFO 패턴에 따라 동작
* 직접적인 응용 : 순서화 시뮬레이션의 대기열, 통신에서 데이터 패킷들의 전송패턴 모델링, 프린터와 컴퓨터 사이의 버퍼링
* 간접적인 응용 : 스택과 마찬가지로 프로그래머의 자료저장/검색/조회 등의 도구, 많은 알고리즘에서 응용되고 있음
* 배열로 구성하는 큐 : 선형 큐, 원형 큐(공백상태, 포화상태)

[7장] 트리
* 트리 : 계층적인 구조를 나타내는 자료구조, 2차원적인 시각패턴
* 구성 : 상위자료(부모)-하위자료(자식) 간의 관계 순서로 저장 및 탐색
* 응용분야 : 계층적 조직구조 및 검색, 파일 시스템 구조 및 검색, 인공지능에서의 검색 등
* 주요 용어 : 노트, 루트. 서브트리, 단말노드, 비 단말노드, 레벨, 높이, 치수
* 이진 트리 : 노드가 최대 2개의 서브 트리를 가지는 트리 (노드에 연결된 자식 노드의 개수는 최대 2개)
* 이진 트리의 분류 : 포화 이진 트리(트리의 각 레벨에 노드가 꽉 차있는 형태) / 완전 이진 트리 : 높이가 h일 때 모든 레벨의 노드가 모두 채워져 있고, 마지막 레벨에서는 왼쪽부터 오른쪽으로 순서대로 채워져 있는 이진트리

[8장] 우선순위 큐
* 의미 : 저장된 데이터의 우선순위를 기준으로 저장하고 출력하는 큐
* 응용분야 : 시뮬레이션 시스템, 네트워크의 트래픽 제어, 운영 체제의 작업 스케쥴링 등
* 구현방법 : 배열을 이용한 우선순위 큐, 연결리스트를 이용한 우선순위 큐, 히프트리를 이용한 우선순위 큐
* 히프트리 : 노드의 키 값이 다음 조건을 만족하는 완전 이진트리 / 구현 방법 : 완전 이진트리이므로 각 노드에 번호를 붙일 수 있음 -> 배열의 첨자로 활용 / 삭제 : 최대 히프의 삭제 동작은 가장 큰 키 값을 가진 노드를 출력하는 것을 의미 -> 당연히 루트가 삭제

[9장] 정렬
* 정렬 : 대상을 크기 순으로 나열하는 것 (오름차순 또는 내림차순) / 컴퓨터 분야에서 가장 많이 사용, 기본적이고 중요한 알고리즘 중 하나 / 자료 탐색에 있어서 필수 기능
* 주로 사용하는 주요 정렬 알고리즘 : 삽입정렬, 선택정렬, 버블정렬, 퀵정렬, 히프정렬, 합병정렬, 기수정렬
* 정렬 알고리즘 분류 : 삽입법(삽입정렬, 쉘정렬), 교환법(버블정렬, 퀵정렬), 선택법(선택정렬, 쉐이크정렬), 분할법(합병정렬 등)

[10장] 그래프
* 그래프 : 어떤 연관성에 의해 연결된 객체 간의 관계를 표현하는 자료구조 예) 전기회로, 업무 스케줄, 공정 관리, 네비게이션 맵, 순서도
* 그래프 이론 : 그래프를 문제 해결의 도구로 이용하는 분야
* 오일러 문제 : 모든 다리를 한번만 건너서 처음 출발했던 장소로 돌아오는 문제
* 오일러 경로 : 정점에 연결된 간선의 개수가 짝수이면 원래 지점으로 돌아올 수 있는 경로가 존재
* 그래프 종류 : 무방향 크래프(방향성이 없는 간선을 통해서 양방향으로 갈 수 있음), 방향 그래프(방향성이 존재하는 간선을 가진 그래프, 일방통행 도로처럼 한쪽 방향으로만 갈 수 있음을 의미), 가중치 그래프 (간선에 가중치가 주어져 있는 그래프)
* 그래프 용어 : 차수(정점에 연결된 다른 정점의 개수), 경로(출발점에서부터 도착점까지의 정점의 나열), 경로 길이(두 정점 사이의 경로를 구성하는데 사용된 edge 개수), 완전 그래프(모든 정점이 연결되어 있는 그래프)
* 표현하는 두가지 방법 : 인접 행령, 인접 리스트
* 그래프 탐색 두가지 방법 : 깊이우선탐색, 너비우선탐색
* 최소비용 신장트리 : 모든 정점들을 연결한 신장트리의 간선들이 가진 값이 최소인 신장트리
* MST 알고리즘 : Kruskal 알고리즘과 Prim 알고리즘
